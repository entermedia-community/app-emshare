Phase 1: Gather Your Files

  • Upload some files

    The upload area will default to creating an Album for each user. You can tag your files with specific metadata or data entities. Click on 'Pick Files'.

  • Create additional Data Entities

    An Entity is a tag such as a "products", "location", "project" or "event" that you might use to relate to your assets. Click the '+Add New' button and create a new Location 'Entity'. To create a new Entity type add a New Module and set up as "Is Entity"

  • Upload more files and keep the folder structure

    You can pick multiple files by holding the Control/Command key. Or choose a folder structure that will be uploaded into your album.Click on 'Pick Folder'.

  • Edit Metadata

    You can click on a thumb-nail to edit metadata.

  • Multi-file Editing

    You can use the check-boxes or hold the Control/Command key to select multiple thumb-nails, then use the action menu to edit multiple assets at once.
