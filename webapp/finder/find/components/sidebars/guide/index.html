Phase 1: Gather Your Files

  • Create an Album to Upload into

    Requires that each user have an Album. If an Album is not selected, one will be created one for you.

  • Create Entities

    An Entity is a tag such as a "products", "location", "project" or "event" that you might use to relate to your assets.

  • Upload files

    You can pick multiple files by holding the Control/Command key. Or choose a folder structure that will be uploaded into your album.

  • Edit Metadata

    You can click on a thumbnail to edit metadata.

  • Multi-file Editing

    You can use the checkboxes or hold the Control/Command key to select multiple thumbnails, then use the action menu to edit multiple assets at once.

  • Use our Support chat

    If you get stuck at any point, just chat with our Support staff. We'll help you get all setup.

Phase 2: Expand

    Single Sign (SSO), indexing network drives, Amazon Elemental, Amazon S3 sync, Google Drive sync, YouTube uploading, WordPress Drupal, Wowza, etc.
