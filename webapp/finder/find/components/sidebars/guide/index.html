[[Creating an Easy to Use Library]]

Phase 1: Organize your files with Smart Folders

  • Step 1: Setup User Friendly Theme

    1.1: Use the theme editor to change logo or colors

    1.2: Can customize any part of the UI using the Debug Developer Mode

  • Step 2: Creating Smart Folders to Track Everything

    2.1: Use the Smart Organizer to create a new Smart Folder based on your desired storage layout and terminology, e.g.; "Products", "Locations", "Projects" , "Events", etc.

    2.2: Enter a label and choose an helpful Icon

    2.3: Or Pre-made Smart Folders can be downloaded or exported from an existing site

    2.4: The top tab menu layout, located under the main search bar, is arranged by the Smart Folders. Set the order of the tabs by double clicking on the Smart Folder and choosing the menu position

    ##

    Watch Video:Open

  • Step 3: Use Smart Folder Relationships

    3.1: Smart Folders can be connected to one another to illustrate your workflow. Draw a connection from left to right between Smart Folders, creating a one-to-many relationship

    3.2: You can also have field relationships between folders as a linked field. So there is no limit.

    3.3: Enable Lightboxes to further flush out your folder structures

  • Step 4: Setup Data Fields

    4.1: Double on Smart Folder to click on Settings, Editing Fields and Search Fields

    4.2: Or Click on settings to edit Permissions and Fields can be customized

    4.3: Customize your Table results, Add New, Data and Summary panels

Phase 2: Centralize Your Media

  • Step 6: Uploading With a Browser

    6.1: Using the "Files" side icon and right click on any folder

    6.2: Or use the "Upload To:" button on the top of the page to create folders or pick existing folder

    6.3: Or use the "Create New" button and Click the "Files" tab. There is an upload button there

    6.4: Or Click Import button on any entity and see the options

  • Step 7: Bulk Folder Importing

    7.1: Download and Install Desktop app

    7.1:Do a bulk import using the Cloud Sync icon on the right side

    7.3:Or Create a new Folder using "Create New" then use the Import Button

  • Step 8: Online Image Cropping

    8.1:Click on an image expand buttons and click on Image Editor

    8.2:Crop online then use the Save As to create a second image. The two images are linked together

    8.3:Crop Or edit the image and save with the same name. A new version will be created under the "More | History" areas

  • Step 9: Light box Asset and Folder Editing

    9.1: Use the light boxes to drag and drop images from the Working Folders

    9.2: Press the "Open Locally button" to edit the images on your computers and save

    9.3: Press the Sync button to complete the save. New revisions or versions will be made in each Lightbox with a link to the original

  • Step 10: Enable Hot Folders

    10.1: Local storage drives, Cloud: Google Drive, Amazon or FTP volumes

    10.2: Track all edits and deletes.

    10.3: Automatically create Smart Folders.

Phase 3: Efficient editing and processing of media files

  • Step 11: Setup Workflows with Lightboxes

    11.1: Organize the sub-folders within your most important smart folders.

    11.2: Folders can be customized by data type

    11.3: Use Notifications on each folder activity

  • Step 12: History and Version Control

    12.1: Organize the sub-folders within your most important smart folders.

    12.2: On the bottom of each detail there is a Settings, Editing Fields and Search Fields

  • Step 13: Add Status Fields

    13.1: Add extra fields of each Smart Folder

    13.2: Create a production status field with a drop of values and add it to the organizer

  • Step 14: Asset Approval

    14.1: Hide Assets from searches that are pending

    14.2: Finally, use asset Approval to do final approval or rejection notifications

Phase 4: Intuitive Searching

  • Step 15: Smart Folder Categorization

    15.1: Search Categories can be applied to any Smart Folder and searched

    15.2: Helpful on the home page for end user searching

  • Step 16: Automatic Facial Recognition

    16.1: Uses internal search engine to combine similar faces

  • Step 17: AI Tagging

    17.1: By default uses Google AI to categorize images

    17.2: Will also pull all the metadata from the files themselves for searching

  • Step 18: Folder Summary Panel

    18.1: Quick way to use checkboxes and limit results

  • Step 19: Multiple Folder Relationships (Linking)

    19.1: By traversing relationship links you can find connected records

    19.2: Create one to many or one to one relationship fields
