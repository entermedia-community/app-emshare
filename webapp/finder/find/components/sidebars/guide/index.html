Phase 1: Organize your files with Smart Folders

  • Step 1: Setup User Friendly Theme

    1.1: Use the theme editor to change logo or colors

    1.2: Can customize any part of the UI using the Debug Developer Mode

  • Step 2: Creating Smart Folders to Track Everything

    2.1: Use the Smart Organizer to create a new Smart Folder based on your desired storage layout and terminology, e.g.; "Products", "Locations", "Projects" , "Events", etc.

    2.2: Enter a label and choose an helpful Icon

    2.3: Or Pre-made Smart Folders can be downloaded or exported from an existing site

    2.4: The top tab menu layout, located under the main search bar, is arranged by the Smart Folders. Set the order of the tabs by double clicking on the Smart Folder and choosing the menu position

  • Step 3: Use Smart Folder Relationships

    3.1: Smart Folders can be connected to one another to illustrate your workflow. Draw a connection from left to right between Smart Folders, creating a one-to-many relationship

    3.2: You can also have field relationships between folders as a linked field. So there is no limit.

    3.3: Enable Lightboxes to further flush out your folder structures

  • Step 4: Setup Data Fields

    4.1: Double on Smart Folder to click on Settings, Editing Fields and Search Fields

    4.2: Or Click on settings to edit Permissions and Fields can be customized

    4.3: Customize your Table results, Add New, Data and Summary panels
