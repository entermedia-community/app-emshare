[[Custom Links]]

[[Image links can be dynamically created using this format]]:

[server name] [applicationid id] [/converted/cache/] [sourcepath] [file name] [extension] [?]


[[Example to create a small PDF download]]:
$context.siteRoot$home$apphome/views/modules/asset/downloads/converted/cache/$asset.sourcepath/filename.pdf?prefwidth=1024

extension: [[Can be any supported output format such as]] pdf, jpg, png
prefwidth=1024 [[How wide the image should be in pixels]]
prefheight=300 [[How tall the image should be in pixels]]
pagenum=1: [[Requested page number. Only used with multi-page image formats such as PDF]]
canforcewatermark=true [[Adds a watermark]]
